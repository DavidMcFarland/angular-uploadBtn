angular-uploadBtn

A custom directive that gives you a button when clicked prompt for the file and uses XHR2 to post the file to the server using the action attribute as the endpoint.

When complete, the directive fires the function set to the complete attribute.

Usage

<upload-button
  action="api/file"
  complete="uploaded($data, $status)">
  Upload File
</upload-button>

The server posts the file under the field uploadFile.

Server Example

app.post('/api/file', function(req,res) {
  console.log(req.files.uploadFile);
  res.send();
});

Install

bower install angular-upload-button

<script src="/bower_components/angular-uploadBtn/upload-button-btn.js"></script>

angular.module('App', ['upload.button']);
